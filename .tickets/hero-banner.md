# Hero banner + richer visual design (typography, depth, ESPN-inspired)

Current design reads flat/monotone: pure-black background, generic heading font, no hero. Make it genuinely attractive (ESPN.com / live-tennis.eu reference, but better). Scope:
1. TOP HERO BANNER per main page: a gradient band (per-sport accent -> dark) with the sport icon, large display title, LIVE indicator, and a key-stat strip (e.g. current #1 + points, # live now). Not just a small text title.
2. BACKGROUND DEPTH: move off flat #0A0B0F to a layered/gradient dark (subtle radial/linear gradient, elevated surfaces) so it doesn't feel like a black void. Keep light theme tasteful too.
3. TYPOGRAPHY: add a distinctive sporty display font for headings (NOT Inter/Roboto/Arial) with strong weight/scale; keep a clean body font. Improve hierarchy.
4. Apply across home + ATP/WTA live + Top-1000 + World Cup; keep readability (dark+light) and per-sport accent.

## Acceptance Criteria

Each main page has an attractive hero banner with key stats; background has visual depth (not flat black); headings use a distinct display font; cohesive + readable in dark and light; build+lint+readability green; verified live.
