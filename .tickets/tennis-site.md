# Tennis live-rankings site to rival & surpass live-tennis.eu

Build a live ATP/WTA rankings site that rivals live-tennis.eu in quality and surpasses it in ad revenue. Data from ESPN public API. Deployed on Vercel.

## Design

Agentic build loop per Thariq's dynamic-workflows article: each feature built then checked by an INDEPENDENT adversarial verifier agent (worker != judge). Engineer against agentic laziness, self-preferential bias, and goal drift. Loop-until-done with explicit bounds; pair /loop (cadence) + /goal (hard completion).
