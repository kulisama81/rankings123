# Tennis streak and form indicators

Show recent match win/loss streak (W5, L3, etc.) and recent form (last 10 matches) for each player in rankings

## Acceptance Criteria

ATP/WTA live tables show streak indicator (e.g., W5 = won last 5, L2 = lost last 2) and form sparkline or text (e.g., WWLWW); tooltip shows recent matches; data from ESPN scoreboard/results; no fabricated streaks (show only when data available)
