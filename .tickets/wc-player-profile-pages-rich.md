# World Cup: Enhanced player profile pages (career, club, tournament stats)

Build rich player profile pages with tournament performance, career history, club affiliation, and comparative analytics. SofaScore, FotMob, ESPN all have detailed player pages. Currently basic pages with only tournament goals/assists/appearances. Rich profiles = SEO engine (player name searches) and engagement depth.

## Acceptance Criteria

✓ Player profile pages at /world-cup/player/[playerId]
✓ Tournament stats: games, goals, assists, minutes, yellow/red cards
✓ Career overview: caps, career goals, debut date
✓ Current club with logo and league
✓ Position and date of birth
✓ Performance chart (goals/assists by tournament match)
✓ Team affiliation (link to team page)
✓ Data from ESPN or FIFA source (with mock fallback)
✓ Mobile responsive
✓ SEO optimized for player name searches
✓ OpenGraph for social sharing

**Impact:** HIGH — SEO engine + engagement depth
**Effort:** MEDIUM-HIGH — data modeling + page template
**ROI:** VERY HIGH — long-tail SEO traffic from player name searches
