# Sport icon glyphs in navigation tabs

Add distinctive sport icons/glyphs to nav tabs (ATP, WTA, World Cup, Cycling) to strengthen visual identity and scannability. Move beyond text-only nav.

## Acceptance Criteria

- Each nav tab has a matching sport icon/glyph
- Icons are cohesive (same style, weight, size) - NOT emoji, use SVG or icon font
- Icon + label layout: side-by-side on desktop, icon-only on mobile <640px
- Icon color inherits from accent when tab is active
- Accessible (aria-label for icon-only mobile views)
- Adds to the icon-system initiative (not one-off)
