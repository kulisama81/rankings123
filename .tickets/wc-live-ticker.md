# World Cup live match ticker

Real-time goal notifications / event feed across all live matches (homepage + WC pages)

## Acceptance Criteria

Ticker shows goals/red cards/penalties across all live matches with timestamps; auto-updates (polling or SSE); dismissible; mobile-friendly; no fabricated events; only shows when matches are actually live

## Notes

**2026-06-26T13:48:58Z**

ROI: Peak engagement during live match windows (multiple games simultaneously). First principles: users checking rankings during live games want to know 'what just happened' without leaving the page — a live ticker captures attention and drives session time.
